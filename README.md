# TheNexus

## Starting the Frontend
```{text}
cd frontend
```
### If it is your first time running the frontend
```{text}
npm install
```
```{text}
npm run dev
```

## Starting the Backend
```{text}
cd backend
```
### If you haven't created a python virtual envioment
```{text}
python -m venv venv
```
```{text}
source venv/bin/activate
```
```{text}
uvicorn main:app --reload
```
